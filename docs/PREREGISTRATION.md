# SEDE — Pre-registered predictions

**Registered:** 2026-06-24. The git commit adding this file (its hash + timestamp) is the
immutable pre-registration record — every number below is committed *before* the target data
(DESI DR3, Euclid, LVK O5, …) are in hand.

**Why SEDE is pre-registerable.** The canonical model is **parameter-free in the dark sector**:
```
ρ_DE ∝ H^{2λ} f_sat(z),   λ = 1 − Δ/2 = 0.5   (Barrow Δ = 1, derived: Thms 8–9),
f_sat(z) = (1 − e^{−γ D²}) / (1 − e^{−γ}),     γ = 1.4964   (Sheth–Tormen, fixed).
```
There are **no free dark-energy knobs** — only the 5 parameters ΛCDM also has (Ω_m, H₀, ω_b,
M_B, σ8). So the predictions cannot be tuned after the fact. Numbers generated by
`gen_predictions.py` → `predictions.json` at the best-fit **Ω_m ≈ 0.30, H₀ ≈ 67.5** (values that
depend on Ω_m are flagged; they shift with Ω_m via the stated relation, not as free choices).

Adjudication rule: a prediction is **falsified** if the measured value violates the stated
criterion at the indicated significance. SEDE is *confirmed* on a given test if the data land on
its committed value and away from ΛCDM. "Much less wrong, not right" caveat: current fits give
χ²/dof≈3 against DESI's internal BAO tension — the tests below are about SEDE *vs* ΛCDM, not
absolute goodness of fit.

## E. Committed CMB validation follow-ups (triggered, not predictions)

The CMB analysis uses the compressed (R, ℓ_A) priors in the joint, validated against the **full primary
plik_lite TTTEEE + lowℓ** likelihood — marginalised (E1, `run_full_cmb_mcmc.py`, Δχ²(SEDE−ΛCDM) = −0.43)
and folded into the joint (`run_joint_fullcmb.py`). Two heavier likelihoods are **not installed**; we
commit here to running them on explicit triggers, so the scope is pre-declared, not silently dropped:

| follow-up | what it adds | trigger to run it |
|---|---|---|
| **Full non-lite `plik` (sampled foregrounds)** | lets the ~20 Planck foregrounds float instead of being marginalised | a referee requests it, OR the `run_joint_fullcmb` profile shifts the preference by >1 in Δχ² vs the compressed-CMB joint |
| **ACT DR6 *primary* (multifrequency TT/TE/EE)** | an independent instrument's peaks (the Wu et al. 2509.02945 HDE-killer used ACT primary) | a referee requests it, OR before resubmission if time permits |

*Committed expectation (falsifiable):* both are expected to give |Δχ²(SEDE−ΛCDM)| < 1, consistent with
the plik_lite primary (−0.43) and ACT DR6 *lensing* (−0.31 at the best-fit), because SEDE has *less*
early dark energy than ΛCDM (Ω_DE(z=3) ≈ 0.035 < 0.040), not the runaway high-z growth that breaks
standard HDE. A result with a *large positive* Δχ² in either would be a genuine failure.

> **Status update (post-registration).** The full non-lite `plik` trigger has since **fired and been

## F. Structure-sourcing mechanism test (E1) — pre-registered decision rule

This is the decider for SEDE's **distinctive** claim (dark energy sourced by structure growth), as
opposed to its EOS shape (which is CPL-degenerate). It is registered here as an exact, locked rule so
"staged" carries no post-hoc freedom. Pipeline: `run_e1_mechanism.py` (implemented, runs now;
RSD-limited at z>1 with current data → inconclusive, χ²/dof = 0.40 RSD-budget / 1.40 geometry-budget).

**Data vector (committed).** Two *independent* reconstructions of the activation fraction f_sat(z):
- **Geometry leg** f_sat^geom(z): from the expansion history — DESI DR3 BAO D_H(z)/r_d and D_M(z)/r_d
  → H(z) → ρ_DE(z) → f_sat^geom = ρ_DE/(ρ_DE0 · E^{2λ}), λ = 0.5.
- **Growth leg** f_sat^grow(z): from structure — Euclid + Rubin/LSST weak-lensing **3×2pt** tomography
  (cosmic shear + galaxy–galaxy lensing + clustering, ≥5 source bins to z≈2) + CMB-lensing cross-spectra
  + cluster abundance + RSD fσ8, giving D^grow(z) and Σ_S → f_sat^grow.

**Test statistic.** The parameter-free null r(z) ≡ ρ_DE^geom(z) − G[D^grow(z)] = 0 across the
tomographic bins, equivalently the growth-side deformation Δ_grow measured against the geometry-side
Δ_geom (forecast joint σ(Δ)_grow = 0.13; `run_mechanism_forecast.py`).

| outcome | criterion (committed) | meaning |
|---|---|---|
| **CONFIRM** SEDE mechanism | \|Δ_grow − Δ_geom\| < 2σ_joint **and** the null r(z)=0 has χ²/dof ≲ 1 across the z-bins, **and** Δ_grow = 1.0 ± 0.13 | f_sat from growth = f_sat from geometry ⇒ DE *does* track structure |
| **REFUTE** structure-sourcing | Δ_grow consistent with **0 at >3σ** while Δ_geom = 1 (geometry needs DE, growth does not source it), **or** r(z) ≠ 0 at >3σ with the ΛCDM (no-D-dependence) shape | DE is not sourced by growth — SEDE falsified as a *mechanism* (may survive as a w(z)) |
| **distinguish rivals** | ΛCDM: ρ_DE = const ⇒ r(z) ≠ 0, monotonic specific shape; Gough IDE: ρ_DE = G′[SMD_baryonic] ⇒ diverges from the gravitational D(z) curve by ~50% at z=3 | the same null separates SEDE from both ΛCDM and the star-formation-sourced rival |

*Committed expectation:* SEDE predicts CONFIRM at ∼4σ with full Euclid/LSST tomography (σ(Δ)_grow→0.13,
§6); current data cannot decide (RSD-limited). The criteria above are fixed as of this commit.

## The single make-or-break
If forced to one line: **DESI DR3 + Euclid will measure Δ; SEDE says Δ = 1.0 ± 0.09 and the EOS
crosses −1 with w₀ ≈ −0.98, w_a ≈ −0.11.** A measurement of Δ consistent with 0 (a smooth,
Bekenstein–Hawking horizon), or a CPL point sitting at ΛCDM (−1, 0), falsifies the parameter-free
SEDE-H. The Fisher forecast (`run_tier2_forecast.py`) says these surveys separate Δ=1 from Δ=0 at
~11σ — so this is a genuine, near-term, pre-registered decider.
